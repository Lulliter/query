# OIS

Quarto website for OIS project.

## Structure

- `index.qmd` - Home page
- `pp.qmd` - PP page
- `bandi.qmd` - Bandi page
- `data_in/` - Input data (create subfolders as needed)
- `data_out/` - Output data (create subfolders as needed)

## Usage

Render the website:

```bash
quarto render
```

Preview locally:

```bash
quarto preview
```
